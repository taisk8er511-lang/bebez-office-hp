
# べべつ事務所 HP - プロジェクトガイド

## 🎯 プロジェクト概要

チャットボット機能付きのモダンで洗練されたホームページ開発プロジェクトです。
訪問者とリアルタイムで対話できるポートフォリオサイトを目指しています。

---

## 📋 コーディング規約

### 言語設定
- **返答は日本語で行う**
  - コメント、エラーメッセージ、ドキュメントはすべて日本語
  - ユーザーへの説明や指示も日本語

### HTML/CSS/JavaScript
- セマンティックなHTML構造を心がける
- CSSはBEM命名規則に従う（例: `block__element--modifier`）
- JavaScriptはES6+の構文を使用
- コードは簡潔で読みやすい状態を保つ

### コメント規約
- **すべてのコメントは日本語で記述**
- 複雑なロジックやビジネスロジックには説明的なコメントを付与
- 関数には用途・引数・戻り値の説明を含める
- 例：
  ```javascript
  // ユーザーのメッセージをチャットボットに送信
  // @param {string} message - ユーザーが入力したメッセージ
  // @returns {Promise<string>} - チャットボットの返答
  ```

## 🎨 デザイン原則

### ビジュアル
- **モダンで洗練されたデザイン**を常に心がける
- アクセントカラー：**うぐいす色（#6B8E23）**を基調
- 白を背景色とする（視認性とシンプルさの両立）
- グラデーション・シャドウを上品に使用

### ユーザーインターフェース
- 最小限のシンプルさ（不必要な装飾は避ける）
- ホワイトスペースを有効に活用
- マイクロインタラクション（ホバー時の視覚効果など）を丁寧に設計

### カラーパレット
```
プライマリ（うぐいす色）: #6B8E23
ダーク（濃いうぐいす）: #556B2F
背景色（白）: #ffffff
テキスト（濃いグレー）: #333333
サブテキスト（薄いグレー）: #666666
背景サポート（薄いグレー）: #f9f9f9
```

## 📱 レスポンシブ対応

### 必須仕様
- **すべてのページはモバイルファーストで設計**
- スマートフォン（320px～）対応は必須
- タブレット（768px～）対応は必須
- デスクトップ（1200px～）対応は必須

### ブレークポイント
```css
/* スマートフォン: 320px以上 */
/* タブレット: 768px以上 */
/* デスクトップ: 1200px以上 */
```

### 要件
- タッチターゲットのサイズ（最小44x44px）を確保
- フォントサイズ：最小16px（ズーム時の誤操作防止）
- 画像は `max-width: 100%` で自動スケール
- フレックスボックス・グリッドレイアウトを活用
